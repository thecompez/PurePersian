# گنجینهٔ واژه‌های فارسی

**نسخهٔ گسترش‌یافته**

این طرح‌نامه شامل مجموعه‌ای از واژه‌های فارسی خالص به همراه معادل‌های عربی‌زده و انگلیسی آن‌هاست. هدف این طرح‌نامه فراهم کردن یک منبع قابل استفاده برای علاقه‌مندان به زبان فارسی، نویسندگان، دانشجویان، برنامه‌نویسان و فعالان فناوری است تا نوشتار و گفتار فارسی روان و خالص داشته باشند.

---

## ویژگی‌ها

- جایگزین‌های فارسی خالص برای واژه‌های عربی‌زده و تخصصی
- ارائه معادل انگلیسی برای استفاده بین‌المللی
- توضیح کاربرد هر واژه برای درک بهتر
- دسته‌بندی واژه‌ها در زمینه‌های مختلف:  
  - واژه‌های تأکیدی و نفی  
  - واژه‌های تقریبی و مثالی  
  - واژه‌های زمان و مکان  
  - واژه‌های اجتماعی و اداری  
  - واژه‌های فکری و انتزاعی  
  - واژه‌های روزمره و طبیعت  
  - واژه‌های آموزشی و علمی  
  - واژه‌های اقتصادی و مالی  
  - واژه‌های فناوری، بلاک‌چین و برنامه‌نویسی  
  - واژه‌های احساسی و روانی  
  - واژه‌های مذهبی و باورها  
  - واژه‌های نرم‌افزاری و رابط کاربری

---

## ساختار طرح‌نامه

- فایل اصلی PDF/LaTeX با جدول‌های چهار ستونه:  
  **عربی‌زده | فارسی خالص | انگلیسی | توضیح کاربرد**
- تمام متن به فارسی است و تنها جاهایی که معادل انگلیسی نیاز بوده اضافه شده است.
- طرح‌نامه به صورت قابل گسترش طراحی شده و کاربران می‌توانند واژه‌های جدید اضافه کنند.

---

## استفاده

1. فایل PDF را باز کنید و واژه‌ها را مرور کنید.
2. از معادل‌های فارسی خالص در نوشتار و گفتار خود استفاده کنید.
3. برای اضافه کردن واژه‌های جدید، می‌توانید فایل LaTeX را ویرایش کرده و Pull Request ارسال کنید.

---

## مشارکت

این طرح‌نامه برای ارتقاء زبان فارسی و غنای واژگان آن طراحی شده است. شما می‌توانید با پیشنهاد واژه‌های جدید، اصلاح ترجمه‌ها و بهبود توضیحات کاربرد، در غنی‌تر شدن این مجموعه مشارکت کنید.
